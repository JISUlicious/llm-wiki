## Concepts

- [[chain-of-thought-prompting]] — Eliciting step-by-step reasoning via few-shot exemplars; emergent at ~100B scale
- [[react]] — Paradigm for interleaving reasoning traces and actions in LLM prompting
- [[reflexion]] — Verbal reinforcement learning: agents learn from failure via self-reflection and episodic memory
- [[tool-use-in-llms]] — Broader paradigm of LLMs interfacing with external tools and environments
- [[agentic-ai]] — Goal-directed AI systems that reason, act, remember, and coordinate with humans or other agents
- [[society-of-thought]] — View that strong reasoning can emerge from internal multi-perspective deliberation
- [[transformer-architecture]] — Attention-first sequence model that displaced recurrent and convolutional seq2seq backbones
- [[self-attention]] — Mechanism where each token attends to other positions in the same sequence
- [[masked-language-modeling]] — Self-supervised objective that predicts masked tokens from bidirectional context
- [[generative-pretraining]] — Autoregressive language-model pre-training followed by downstream fine-tuning
- [[in-context-learning]] — Adapting behavior from prompt instructions and demonstrations without weight updates
- [[dense-passage-retrieval]] — Dual-encoder dense retrieval method that outperforms BM25 for open-domain QA
- [[functional-emotions]] — Emotion-like patterns in LLMs mediated by abstract representations; causally influence alignment-relevant behavior
- [[emotion-vectors]] — Linear representations of emotion concepts in LLM activation space
- [[activation-steering]] — Modifying model activations at inference to control behavior via learned concept directions
